# PRO3600

Plateforme de backtesting PRO3600.

## Description

PRO3600 est une plateforme de backtesting permettant de tester et d'évaluer des stratégies de trading sur des données historiques.

## Fonctionnalités

- Chargement et traitement de données historiques
- Définition et exécution de stratégies de trading
- Calcul de métriques de performance (rendement, drawdown, ratio de Sharpe, etc.)
- Visualisation des résultats

## Installation

```bash
git clone https://github.com/NeilKiuc/PRO3600.git
cd PRO3600
```
